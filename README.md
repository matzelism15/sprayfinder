# Sprühfarben-Finder

Vom Entwurf zur richtigen Dose — eine Browser-App, die ein Bild analysiert und passende
Sprühdosen von **Montana Cans**, **Montana Colors (MTN)**, **Molotow Premium** und
**Loop Colors** vorschlägt.

**👉 Direkt im Browser öffnen:** https://matzelism15.github.io/sprayfinder/

## Was die App macht

- Bild per Drag & Drop oder Datei-Auswahl hochladen
- Wichtigste Farben werden automatisch erkannt (2–16 Zielfarben einstellbar)
- Vorschau zeigt, wie das Motiv in echten Dosenfarben aussieht
- Prozentuale Verteilung jeder Farbe wird angezeigt
- Einkaufsliste mit Hersteller, Farbname und Code
- Einzelne Farben lassen sich manuell durch andere Dosenfarben ersetzen
- Vorschau als PNG exportierbar

Alles läuft **komplett lokal im Browser** — keine Daten werden hochgeladen, keine
Anmeldung nötig, funktioniert auch offline.

## Lokal nutzen

Falls du die App ohne GitHub Pages nutzen willst:

1. `index.html` herunterladen
2. Doppelklick — fertig

Es gibt keine Abhängigkeiten, kein Build-Schritt, kein Server nötig.

## Hersteller-Datenbanken

Die Farbcodes der vier unterstützten Hersteller sind direkt in `index.html` eingebettet.
Stand der Daten: Frühjahr 2026.
